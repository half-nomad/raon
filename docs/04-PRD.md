# Product Requirements Document (PRD)
# Raon Total Solution B2B Website

**Document Version:** 1.0
**Last Updated:** 2025-11-18
**Status:** Approved

## Executive Summary

(주)라온토탈솔루션은 정유·석유화학 및 조선 산업을 대상으로 30년 경험의 회전기계 및 왕복동 압축기 부품·솔루션을 제공하는 B2B 전문 기업입니다. 본 웹사이트는 기업의 전문성과 신뢰도를 효과적으로 전달하고, 잠재 고객의 견적 문의 및 기술 상담을 유도하여 비즈니스 리드를 창출하는 것을 목표로 합니다.

## Product Vision

### Problem Statement
**문제:** B2B 산업재 분야에서 기업의 전문성, 파트너십, 제품 포트폴리오를 명확하게 전달할 수 있는 디지털 접점이 부족합니다.

**영향:**
- 잠재 고객이 공급사의 신뢰도를 확인하기 어려움
- 제품 검색 및 문의 프로세스가 비효율적
- 엔지니어와 구매 담당자가 필요한 기술 정보를 빠르게 찾지 못함

**현재 상황:** PDF 카탈로그와 오프라인 영업에 의존하고 있으며, 온라인 상에서 기업 정보와 제품군을 체계적으로 제공하는 채널이 없습니다.

### Solution
전문적인 B2B 웹사이트를 통해:
- **산업별 솔루션(Industries)** 과 **제품/브랜드(Products)** 관점을 명확히 분리하여 제공
- 30년 업력, 글로벌 파트너십, 주요 고객사 레퍼런스를 시각화
- 견적 문의 및 기술 상담을 위한 직관적인 CTA 및 문의 폼 제공
- 모바일부터 데스크톱까지 모든 디바이스에서 최적의 사용자 경험 제공

### Success Criteria
이 제품이 성공했다고 판단하는 기준:
- [ ] 월 100건 이상의 견적/상담 문의 리드 확보
- [ ] 페이지 로딩 속도 3초 이내 유지
- [ ] 모바일 사용자 비율 40% 이상 유지
- [ ] 주요 키워드(왕복동 압축기, 실린더 라이너 등) 검색 노출 상위 10위 내

## Target Users

### Primary Users: 플랜트 설비 담당 엔지니어
**누구:** 정유·석유화학 플랜트, 발전소 설비팀/유지보수팀
**니즈:**
- 기존 설비와 호환되는 고품질 Spare Parts 검색
- 긴급 상황 시 신속한 납기 및 기술 지원
- 제품 스펙 및 카탈로그 확인

**페인포인트:**
- 부품 호환성 확인이 어려움
- 공급사의 기술력과 대응 속도를 사전에 파악하기 힘듦
- 도면이나 스펙만으로 견적을 요청하기 어려움

### Secondary Users: 구매 담당자
**누구:** EPC사, 플랜트, 조선소 구매팀
**니즈:**
- 공급사의 신뢰도(연혁, 파트너, 레퍼런스) 빠르게 확인
- 공급 가능한 브랜드와 제품 범위 파악
- 간편한 견적 요청

**페인포인트:**
- 여러 공급사를 비교하며 신뢰도를 판단하는 데 시간이 소요됨
- 제품군이 복잡하여 어떤 브랜드/부품을 공급하는지 불명확

### Tertiary Users: 잠재 파트너 / 의사결정자
**누구:** 해외 제조사, 국내 기술 파트너사, 경영진
**니즈:**
- 라온토탈솔루션의 사업 영역 및 역할 이해
- 핵심 역량과 비전 확인

## User Stories

### Must Have (MVP)
1. **산업별 솔루션 탐색**
   - As a: 플랜트 엔지니어
   - I want to: 내가 속한 산업(정유·석유화학 / 조선)에 맞는 솔루션을 먼저 확인
   - So that: 우리 산업에 특화된 공급사인지 빠르게 판단

2. **제품군 검색 및 필터링**
   - As a: 구매 담당자
   - I want to: 7개 제품군(Compressor, Mixer, Pump 등) 중 필요한 제품 확인
   - So that: 공급 가능한 브랜드와 제품 범위를 파악

3. **견적 문의 (도면/사진 첨부)**
   - As a: 설비 엔지니어
   - I want to: 도면, 부품 사진, 스펙시트를 첨부하여 문의
   - So that: 정확한 견적과 기술 상담을 받음

4. **글로벌 파트너 및 고객사 확인**
   - As a: 구매 담당자
   - I want to: 공식 파트너사(SPXFLOW, NEUMAN & ESSER 등) 및 주요 고객사(SK에너지, S-OIL 등) 확인
   - So that: 공급사의 신뢰도를 검증

5. **회사 연혁 및 CEO 인사말**
   - As a: 의사결정자
   - I want to: 회사의 설립 배경, 성장 과정, 비전 확인
   - So that: 장기 파트너로서 신뢰할 수 있는지 판단

## Core Features

### Feature 1: 산업별 솔루션 페이지 (Industries)
**Description:** 정유·석유화학 / 조선 두 산업군에 대한 전문 솔루션 소개 페이지
**User Value:** 타깃 산업에 특화된 전문성을 명확히 전달하여 신뢰 구축
**Priority:** Must Have

**User Flow:**
1. HOME에서 "Industries" 메뉴 클릭 또는 Hero 섹션 CTA 클릭
2. 산업별 카드(정유·석유화학 / 조선) 확인
3. 관심 산업 카드 클릭 → 상세 페이지 이동
4. 해당 산업의 주요 제공 서비스 및 관련 제품 확인
5. 관련 제품 바로가기 링크 → Products 페이지로 이동

**Acceptance Criteria:**
- [ ] 2개 산업 카드가 명확히 구분되어 표시됨
- [ ] 각 산업별 핵심 소개 문구 및 주요 서비스 리스트 표시
- [ ] 관련 제품으로 바로가기 링크 기능 작동
- [ ] 모바일에서 1열, 데스크톱에서 2열 그리드 레이아웃

### Feature 2: 제품군 페이지 (Products)
**Description:** 7개 제품군(Compressor & Parts, Mixer, Motor, Pump, Bearing, Cylinder Liner, Oil Purifier)을 카드 형태로 나열하고, 각 제품군 클릭 시 상세 설명 Accordion 토글
**User Value:** 공급 가능한 제품 범위를 한눈에 파악하고, 필요한 제품 상세 정보를 빠르게 접근
**Priority:** Must Have

**User Flow:**
1. Products 메뉴 클릭
2. 7개 제품군 카드 목록 확인
3. 관심 제품군 카드 클릭 → Accordion 펼침
4. 제품 설명, 주요 특징, 적용 사례, 브랜드 정보 확인
5. (선택) "문의하기" CTA 클릭 → Contact 페이지로 이동

**Acceptance Criteria:**
- [ ] 7개 제품군 카드가 명확히 표시됨
- [ ] Accordion 토글 시 상세 내용 펼침/닫힘 작동
- [ ] 각 제품군마다 브랜드 로고(placeholder) 및 설명 표시
- [ ] 모바일 1열, 태블릿 2열, 데스크톱 3열 그리드

### Feature 3: 문의 폼 (Contact)
**Description:** 구분 선택, 이름, 회사명, 이메일, 전화번호, 문의 내용, 파일 첨부 기능을 갖춘 문의 폼
**User Value:** 도면, 사진, 스펙시트를 첨부하여 정확한 견적 및 기술 상담 요청 가능
**Priority:** Must Have

**User Flow:**
1. Header CTA ("상담 문의" / "제품 문의") 또는 Contact 메뉴 클릭
2. 문의 폼 페이지 진입 (CTA에 따라 구분 필드 자동 선택)
3. 필수 항목 입력 (이름, 이메일, 전화번호, 문의 내용)
4. (선택) 파일 첨부 (도면, 사진)
5. 개인정보 수집 동의 체크
6. "문의하기" 버튼 클릭 → 이메일 전송
7. 성공 메시지 표시

**Acceptance Criteria:**
- [ ] 모든 필수 항목 유효성 검사 작동
- [ ] 파일 첨부 기능 (최대 10MB, PDF/JPG/PNG)
- [ ] 이메일 발송 성공 시 성공 메시지 표시
- [ ] 개인정보 동의 체크박스 필수

### Feature 4: 파트너 및 고객사 페이지 (Partners)
**Description:** 글로벌 파트너사 로고 슬라이더 + 주요 고객사 로고 그리드
**User Value:** 공급사의 신뢰도를 글로벌 파트너십 및 주요 고객사 레퍼런스로 증명
**Priority:** Must Have

**User Flow:**
1. Partners 메뉴 클릭
2. 글로벌 파트너 로고 슬라이더 확인 (SPXFLOW, NEUMAN & ESSER 등)
3. 주요 고객사 로고 그리드 확인 (SK에너지, S-OIL 등)
4. (선택) 각 파트너/고객사 관련 설명 툴팁 확인

**Acceptance Criteria:**
- [ ] 파트너 로고 슬라이더 autoplay 4초, hover 시 pause
- [ ] 고객사 로고 그리드 2열 (모바일) → 4열 (데스크톱)
- [ ] 각 로고 아래 계약 연도/범위 작은 텍스트로 표시

### Feature 5: 회사 소개 페이지 (Company)
**Description:** CEO 인사말, 연혁 타임라인, 오시는 길
**User Value:** 회사의 설립 배경, 성장 과정, 비전을 전달하여 신뢰 구축
**Priority:** Must Have

**User Flow:**
1. Company 메뉴 클릭
2. CEO 인사말 섹션 확인 (2열 레이아웃: 이미지 / 텍스트)
3. 연혁 타임라인 확인 (2020~2025)
4. 오시는 길 섹션에서 주소, 연락처, 지도 확인

**Acceptance Criteria:**
- [ ] CEO 인사말 2열 레이아웃 (모바일 1열)
- [ ] 연혁 타임라인 수평 레이아웃 (모바일 세로)
- [ ] 지도 placeholder 이미지 표시
- [ ] 주소, 전화, 이메일 정보 명확히 표시

## Non-Functional Requirements

### Performance
- 페이지 로딩 시간: < 3초 (Lighthouse Performance 90+ 목표)
- 이미지 최적화: WebP 포맷, lazy loading
- 모바일 데이터 사용량: 페이지당 < 2MB

### Security
- HTTPS 필수 (Vercel 자동 제공)
- 문의 폼 스팸 방지 (Honeypot 또는 reCAPTCHA)
- 개인정보 수집 동의 체크박스 필수

### Accessibility
- WCAG 2.1 AA 준수 목표
- 모바일 반응형 (320px ~ 1920px)
- 키보드 내비게이션 지원
- 명확한 포커스 상태 표시

### SEO
- 페이지별 Title, Meta Description, OG Tags
- Semantic HTML (h1, h2, nav, main, footer)
- 구조화된 데이터 (JSON-LD) - 회사 정보
- 핵심 키워드: 왕복동 압축기, 실린더 라이너, SPXFLOW, 정유석유화학 부품

## User Experience

### Key Principles
1. **B2B 전문성**: 깔끔하고 신뢰감 있는 디자인
2. **빠른 정보 접근**: 3클릭 이내에 모든 주요 정보 도달
3. **모바일 우선**: 모바일에서도 완벽한 경험 제공

### User Journey
**엔지니어 사용자 예시:**
1. 검색엔진 → HOME 페이지 진입
2. Hero 섹션에서 "산업별 솔루션 보기" 클릭
3. "정유·석유화학" 카드 클릭
4. 관련 제품 "Compressor & Parts" 클릭
5. 제품 상세 확인 후 "견적 문의" CTA 클릭
6. 문의 폼에서 도면 첨부 및 제출

## Success Metrics (KPIs)

### Engagement Metrics
- 페이지뷰: 월 5,000 PV 목표
- 세션 지속 시간: 평균 3분 이상
- 이탈률: < 60%

### Business Metrics
- 문의 폼 제출: 월 100건 이상
- 문의 전환율: 방문자 대비 2% 이상
- 주요 CTA 클릭률: 10% 이상

### Quality Metrics
- Lighthouse Performance: 90+
- Uptime: > 99.9%
- 모바일 사용성: 100점
